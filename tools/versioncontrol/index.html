Version Control

Version control is incredibly important and no matter what language you code in, you will most likely need some form of version control.

To keep it simple, I recommend git although there are some other types of version control out there as well (svn).

It is important to note, that git is the name of the version control system. Where you store all of the files however, is completely up to you.

It is recommended however, that you keep your files stored on an online git repository host - think of this as just a storage space for your code.


Two of the bigger repository hosts are

  • GitHub - Most popular, but comes with only 5 free private repos
  • Bitbucket - Not as popular - but comes with unlimited private repos!
